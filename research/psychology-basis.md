## 1. 大五人格模型 (Big Five / OCEAN)

### 1.1 理论介绍

大五人格模型（Big Five Personality Traits）是目前心理学界最广泛接受的人格理论框架，由Costa & McCrae于1980年代系统化。该模型通过五个维度描述人格特质：

| 维度 | 英文 | 核心特征 | 低分者特点 | 高分者特点 |
|------|------|----------|------------|------------|
| **开放性** | Openness | 对新经验的开放程度 | 务实、传统、保守 | 好奇、创新、艺术敏感 |
| **尽责性** | Conscientiousness | 自律与组织能力 | 随性、灵活、冲动 | 有条理、可靠、勤奋 |
| **外向性** | Extraversion | 社交能量与积极性 | 内向、独立、冷静 | 外向、热情、活跃 |
| **宜人性** | Agreeableness | 人际和谐与合作倾向 | 竞争、批判、直接 | 信任、利他、合作 |
| **神经质** | Neuroticism | 情绪稳定性 | 情绪稳定、平静 | 敏感、焦虑、情绪化 |

### 1.2 经典问题（中文改编版）

#### 开放性 (Openness) - 10题

**正向题（同意得分）：**
1. 我经常对抽象的概念和哲学问题感兴趣
2. 我喜欢尝试新的餐厅、旅行目的地或文化体验
3. 我认为自己是一个富有想象力的人
4. 我欣赏艺术和美学，经常被音乐、绘画或文学作品打动
5. 我对许多不同的话题都有好奇心

**反向题（不同意得分）：**
6. 我更喜欢熟悉的事物，不太愿意尝试新的体验
7. 我认为艺术和诗歌不太重要
8. 我很少思考抽象的问题
9. 我倾向于用传统的方式做事
10. 我对新奇和不同寻常的事物不感兴趣

#### 尽责性 (Conscientiousness) - 10题

**正向题：**
1. 我会制定详细的计划并坚持执行
2. 我是一个非常可靠的人，说到做到
3. 我会把事情做得井井有条
4. 我努力工作以实现我的目标
5. 我会仔细检查工作，确保没有错误

**反向题：**
6. 我有时会拖延任务直到最后一刻
7. 我的书桌或房间经常很乱
8. 我容易分心，难以集中精力完成任务
9. 我不善于管理时间
10. 我经常冲动消费

#### 外向性 (Extraversion) - 10题

**正向题：**
1. 我是聚会的活跃分子，喜欢社交场合
2. 我很容易与人交谈，即使是陌生人
3. 我喜欢刺激和兴奋的活动
4. 我通常感觉精力充沛、充满热情
5. 我更喜欢和很多人在一起，而不是独处

**反向题：**
6. 我更喜欢独自工作或小团体活动
7. 大型社交场合让我感到疲惫
8. 我是一个安静、内向的人
9. 我很少主动发起对话
10. 我需要大量独处时间来恢复精力

#### 宜人性 (Agreeableness) - 10题

**正向题：**
1. 我相信大多数人是善良诚实的
2. 我愿意帮助需要帮助的人
3. 我倾向于信任他人而不是怀疑他们
4. 我努力避免与他人冲突
5. 我对别人的感受很敏感

**反向题：**
6. 我有时会利用他人来达到自己的目的
7. 我认为在必要时刻与他人竞争是重要的
8. 我不怕与他人对立
9. 我更关心自己的利益而不是别人的
10. 我倾向于批判性地分析他人的动机

#### 神经质 (Neuroticism) - 10题

**正向题（高分=高神经质）：**
1. 我经常感到担忧或焦虑
2. 我的情绪波动比较大
3. 我很容易感到压力
4. 我有时会感到沮丧或低落
5. 我对批评比较敏感

**反向题（低分=情绪稳定）：**
6. 我通常很冷静，不容易生气
7. 我能很好地应对压力
8. 我很少感到焦虑或紧张
9. 我的情绪很稳定
10. 我不容易被事情困扰

### 1.3 计分方法

**李克特量表：**
- 1 = 非常不同意
- 2 = 不同意
- 3 = 中立
- 4 = 同意
- 5 = 非常同意

**计算方式：**
```
各维度得分 = (正向题得分之和 + (6 - 反向题得分之和)) / 题目数量
最终得分范围：1-5分
```

**标准化建议：**
- 可转换为百分位数或T分数（均值50，标准差10）
- 建议建立中文常模数据库

### 1.4 约会匹配应用

**匹配策略：**

| 维度 | 相似性建议 | 互补性建议 |
|------|------------|------------|
| 开放性 | **相似最佳**：差异太大可能导致生活方式冲突 | 适度差异可增加新鲜感 |
| 尽责性 | **相似最佳**：计划型vs随性型易产生矛盾 | - |
| 外向性 | 可适度互补 | 内向+外向可形成平衡 |
| 宜人性 | **高宜人性匹配更和谐** | 低宜人性组合可能冲突多 |
| 神经质 | **情绪稳定者更佳** | 两高神经质易情绪共振 |

**关键发现：**
- 尽责性和宜人性对长期关系质量预测力最强
- 开放性相似度影响共同兴趣和生活方式
- 神经质是关系稳定性的负向预测因子

## 2. MBTI性格测试

### 2.1 理论介绍

MBTI（Myers-Briggs Type Indicator）基于荣格心理类型理论，由Katharine Cook Briggs和Isabel Briggs Myers开发。将人格分为16种类型，由四个维度组合而成：

| 维度 | 偏好一 | 偏好二 | 核心差异 |
|------|--------|--------|----------|
| 能量来源 | E（外向 Extraversion） | I（内向 Introversion） | 获取能量的方式 |
| 信息获取 | S（感觉 Sensing） | N（直觉 Intuition） | 关注现实vs可能性 |
| 决策方式 | T（思考 Thinking） | F（情感 Feeling） | 理性vs感性决策 |
| 生活态度 | J（判断 Judging） | P（感知 Perceiving） | 计划vs灵活 |

### 2.2 经典问题（中文改编版）

#### E/I 维度（外向/内向）- 8题

1. 在社交聚会中，你通常：
   - A. 主动与很多人交谈，享受热闹氛围（E）
   - B. 与少数人深入交流，或提前离开（I）

2. 当你需要恢复精力时，你更倾向于：
   - A. 和朋友外出活动（E）
   - B. 独自在家放松（I）

3. 在思考问题时，你通常：
   - A. 通过与人讨论来理清思路（E）
   - B. 需要先独自思考再做决定（I）

4. 你给人的印象是：
   - A. 热情、健谈、容易接近（E）
   - B. 安静、内敛、需要时间了解（I）

5. 你更喜欢：
   - A. 广泛的社交圈（E）
   - B. 几个深交的朋友（I）

6. 在团队中，你通常：
   - A. 积极发言，推动讨论（E）
   - B. 先倾听，在适当时发言（I）

7. 你的理想周末是：
   - A. 参加派对或社交活动（E）
   - B. 在家看书或追剧（I）

8. 你说话的方式倾向于：
   - A. 快速表达，边说边想（E）
   - B. 三思后言，言简意赅（I）

#### S/N 维度（感觉/直觉）- 8题

1. 你更关注：
   - A. 当下的现实和细节（S）
   - B. 未来的可能性和整体（N）

2. 阅读时，你更喜欢：
   - A. 事实明确、有具体例子的内容（S）
   - B. 探讨概念、富有想象力的内容（N）

3. 你更信任：
   - A. 经验和已证实的方法（S）
   - B. 直觉和新的可能性（N）

4. 你在描述一件事时：
   - A. 注重细节和事实（S）
   - B. 关注整体印象和意义（N）

5. 你更擅长：
   - A. 处理具体、实际的任务（S）
   - B. 发想创意、解决抽象问题（N）

6. 你更喜欢的工作方式是：
   - A. 按照既定步骤执行（S）
   - B. 自由发挥，探索新方法（N）

7. 你更容易注意到：
   - A. 环境中的具体细节（S）
   - B. 背后的模式或趋势（N）

8. 你认为成功的标准是：
   - A. 达成具体、可衡量的目标（S）
   - B. 实现理想或愿景（N）

#### T/F 维度（思考/情感）- 8题

1. 做重要决定时，你更看重：
   - A. 逻辑分析和客观事实（T）
   - B. 对他人的影响和和谐（F）

2. 你更欣赏哪种品质：
   - A. 理性、公正（T）
   - B. 同情心、善良（F）

3. 当别人向你倾诉时，你更倾向于：
   - A. 提供解决方案（T）
   - B. 给予情感支持（F）

4. 在冲突中，你更关注：
   - A. 什么是正确的（T）
   - B. 维护关系（F）

5. 你认为更好的领导是：
   - A. 公正、客观的决策者（T）
   - B. 关心团队成员感受的领导（F）

6. 你更容易被什么打动：
   - A. 逻辑严密的论证（T）
   - B. 真诚的情感表达（F）

7. 在批评他人时：
   - A. 你会直接指出问题（T）
   - B. 你会考虑对方感受，婉转表达（F）

8. 你认为重要的价值观是：
   - A. 真实、公正（T）
   - B. 和谐、关爱（F）

#### J/P 维度（判断/感知）- 8题

1. 你更喜欢的工作方式：
   - A. 制定计划并按计划执行（J）
   - B. 保持灵活，随机应变（P）

2. 你的旅行风格是：
   - A. 提前规划好行程（J）
   - B. 到了再说，随性探索（P）

3. 你的房间/办公桌通常是：
   - A. 整洁有序（J）
   - B. 有点乱但能找到东西（P）

4. 对于截止日期：
   - A. 我会提前完成（J）
   - B. 我在压力下效率更高（P）

5. 你给人的印象是：
   - A. 有条理、可靠（J）
   - B. 灵活、随和（P）

6. 做决定时：
   - A. 我能快速做出决定（J）
   - B. 我喜欢保持选项开放（P）

7. 你的日程安排：
   - A. 我喜欢有明确的计划（J）
   - B. 太多计划让我感到束缚（P）

8. 项目结束后：
   - A. 我喜欢完成并继续下一个（J）
   - B. 我可能会继续探索新可能（P）

### 2.3 计分方法

**简单计数法：**
- 统计每个维度A和B选项的数量
- 选择数量多的作为该维度的偏好

**示例：**
- E/I: A选项6个 → E
- S/N: B选项5个 → N
- T/F: B选项5个 → F
- J/P: A选项6个 → J
- 结果：ENFJ

**维度强度计算：**
```
强度百分比 = (该偏好得分 / 总题数) × 100%
例：E得分6/8 = 75%（中等偏强的外向偏好）
```

### 2.4 约会匹配应用

**经典匹配理论（Keirsey气质理论）：**

| 类型 | 最佳匹配 | 次佳匹配 | 需要努力 |
|------|----------|----------|----------|
| INTJ | ENFP, ENTP | INTP, ENTJ | SF类型 |
| INTP | ENTJ, ENFJ | INTJ, ENFP | SJ类型 |
| ENTJ | INTP, INFP | INTJ, ENTP | SF类型 |
| ENTP | INFJ, INTJ | INTP, ENTJ | SJ类型 |
| INFJ | ENTP, ENFP | INTJ, INFP | SP类型 |
| INFP | ENFJ, ENTJ | INFJ, ENFP | ST类型 |
| ENFJ | INFP, INTP | INFJ, ENFP | TP类型 |
| ENFP | INTJ, INFJ | ENFJ, INFP | SJ类型 |
| ISTJ | ESFP, ESTP | ISFJ, ESTJ | NF类型 |
| ISFJ | ESFP, ESTP | ISTJ, ESTJ | NT类型 |
| ESTJ | ISFP, ISTP | ISTJ, ESFJ | NP类型 |
| ESFJ | ISFP, ISTP | ESTJ, ISFJ | NT类型 |
| ISTP | ESTJ, ESFJ | ISTJ, ESTP | NF类型 |
| ISFP | ESTJ, ESFJ | ISFJ, ESFP | NT类型 |
| ESTP | ISFJ, ISTJ | ESFP, ESTJ | NJ类型 |
| ESFP | ISFJ, ISTJ | ESTP, ESFJ | NJ类型 |

**应用建议：**
- 相反偏好（E-I, S-N, T-F, J-P）可形成互补
- S/N维度差异是最常见的关系挑战
- N型与N型、S型与S型更易理解彼此

## 3. SLOAN人格模型

### 3.1 理论介绍

SLOAN模型是大五人格的变体，专门为在线约会和社交应用优化。该模型将大五人格的维度重新命名，更直观地应用于人际匹配：

| SLOAN | 对应大五 | 约会应用含义 |
|-------|----------|--------------|
| **S**ocial（社交） | Extraversion | 社交能量和活跃度 |
| **L**eading（领导） | (反向)Agreeableness | 主导性vs顺从性 |
| **O**rganized（有条理） | Conscientiousness | 计划性vs随性 |
| **A**daptive（适应） | (反向)Neuroticism | 情绪稳定性 |
| **N**umeric（理性） | Openness | 理性vs感性思维 |

**类型编码示例：**
- SCOEI = 社交型、顺从型、有条理、情绪稳定、理性型
- RCUEN = 内向型、主导型、随性、情绪不稳定、感性型

### 3.3 计分与类型判定

**得分计算：**
```
各维度得分 = 所有题目得分之和 / 题目数量
得分范围：1-5分
```

**类型字母判定：**
| 得分范围 | 字母 | 含义 |
|----------|------|------|
| 1.0 - 2.4 | R/C/U/E/I | 低分段字母 |
| 2.5 - 3.5 | 中间值 | 需要进一步判定 |
| 3.6 - 5.0 | S/L/O/A/N | 高分段字母 |

**具体对照：**
- Social: R(低) ← 1-2.4 | 2.5-3.5 | 3.6-5 → S(高)
- Leading: C(低/顺从) ← 1-2.4 | 2.5-3.5 | 3.6-5 → L(高/主导)
- Organized: U(低/随性) ← 1-2.4 | 2.5-3.5 | 3.6-5 → O(高/有条理)
- Adaptive: E(低/不稳定) ← 1-2.4 | 2.5-3.5 | 3.6-5 → A(高/稳定)
- Numeric: I(低/感性) ← 1-2.4 | 2.5-3.5 | 3.6-5 → N(高/理性)

### 3.4 约会匹配应用

**匹配原则：**

| 维度 | 相似匹配 | 互补匹配 | 说明 |
|------|----------|----------|------|
| Social | **推荐** | 可能冲突 | 社交需求差异会导致能量不匹配 |
| Leading | 可互补 | **推荐** | 主导型+顺从型易形成平衡 |
| Organized | **推荐** | 易冲突 | 计划vs随性是常见矛盾源 |
| Adaptive | **推荐** | 可能困难 | 两人都低适应性会增加关系压力 |
| Numeric | 可互补 | 可行 | 理性+感性可互补决策风格 |

**类型组合示例：**
- **SCOAI + SCOAI** = 高度相似，易和谐但可能缺乏火花
- **SCOEI + RCUAN** = 高度互补，需要更多理解和磨合
- **SLOAN + RCUEN** = 几乎相反，挑战较大

## 4. 依恋理论问卷

### 4.1 理论介绍

依恋理论（Attachment Theory）由John Bowlby提出，由Mary Ainsworth通过陌生情境实验验证。成人依恋理论由Hazan & Shaver（1987）引入亲密关系研究领域。

**四种依恋类型：**

| 依恋类型 | 自我模型 | 他人模型 | 核心特点 | 比例 |
|----------|----------|----------|----------|------|
| **安全型** | 积极 | 积极 | 信任、亲密、平衡 | ~60% |
| **焦虑型** | 消极 | 积极 | 渴望亲密、害怕被拒绝 | ~15% |
| **回避型** | 积极 | 消极 | 独立、回避亲密 | ~15% |
| **恐惧型** | 消极 | 消极 | 渴望又害怕亲密 | ~10% |

### 4.2 经典问卷（ECR简化版中文改编）

#### 焦虑维度 - 9题

请评价以下陈述与你的符合程度：
（1=非常不同意，2=不同意，3=中立，4=同意，5=非常同意）

1. 我担心会被伴侣抛弃
2. 我希望伴侣能更多地表达对我的感情
3. 当伴侣不在身边时，我会感到不安
4. 我担心伴侣不像我关心他那样关心我
5. 我需要很多确认才能相信伴侣爱我
6. 当伴侣疏远我时，我会感到恐慌
7. 我会过度分析伴侣的言行，寻找他不爱我的迹象
8. 我经常想知道我的伴侣是否真的在乎我
9. 如果伴侣不理我，我会非常焦虑

#### 回避维度 - 9题

请评价以下陈述与你的符合程度：
（1=非常不同意，2=不同意，3=中立，4=同意，5=非常同意）

1. 我更喜欢保持情感上的独立
2. 我对过于亲密的关系感到不舒服
3. 我不希望过度依赖伴侣
4. 我很难完全信任伴侣
5. 当有人试图过于亲近时，我会感到紧张
6. 我不喜欢向伴侣分享内心深处的感受
7. 我更愿意独立解决问题，而不是寻求伴侣的帮助
8. 我有时会觉得伴侣对我的期望太高
9. 我需要大量个人空间

### 4.3 计分与类型判定

**计算方式：**
```
焦虑维度得分 = 焦虑题目得分之和 / 题目数
回避维度得分 = 回避题目得分之和 / 题目数
得分范围：1-5分
```

**类型判定：**

| 焦虑得分 | 回避得分 | 依恋类型 |
|----------|----------|----------|
| 低（<3） | 低（<3） | **安全型** |
| 高（≥3） | 低（<3） | **焦虑型** |
| 低（<3） | 高（≥3） | **回避型** |
| 高（≥3） | 高（≥3） | **恐惧型** |

### 4.4 约会匹配应用

**匹配模式与挑战：**

| 组合 | 动态特点 | 挑战 | 建议 |
|------|----------|------|------|
| 安全+安全 | **最佳组合**：稳定、信任、健康亲密 | 可能缺乏戏剧性激情 | 鼓励共同成长 |
| 安全+焦虑 | 安全能治愈焦虑，提供稳定 | 焦虑方需要学习自我安抚 | 给予确认，建立安全感 |
| 安全+回避 | 安全能给回避方空间 | 回避方需要学习表达需求 | 尊重边界，渐进亲密 |
| 焦虑+回避 | **经典陷阱**：追逐-逃跑循环 | 高冲突，难以满足彼此需求 | 需要专业帮助或高度觉知 |
| 焦虑+焦虑 | 高度情感共鸣 | 情绪共振，相互放大焦虑 | 学习自我安抚，建立边界 |
| 回避+回避 | 低冲突，各自独立 | 缺乏真正亲密 | 刻意练习表达情感 |

**关键发现：**
- 安全型个体几乎与所有类型都能建立良好关系
- 焦虑-回避组合是最常见的不稳定关系模式
- 依恋类型可以通过关系和成长部分改变（"获得性安全感"）

## 5. 价值观量表

### 5.1 Schwartz价值观理论

#### 理论介绍

Shalom Schwartz的价值观理论提出10种普世价值观，呈圆形结构，相邻价值观相容，对角价值观冲突：

| 价值观 | 英文 | 核心内容 |
|--------|------|----------|
| **自我导向** | Self-Direction | 独立、自主、选择自由 |
| **刺激** | Stimulation | 新奇、挑战、冒险 |
| **享乐主义** | Hedonism | 享受、快乐、满足欲望 |
| **成就** | Achievement | 成功、能力、认可 |
| **权力** | Power | 地位、支配、控制资源 |
| **安全** | Security | 稳定、安全、社会秩序 |
| **顺从** | Conformity | 遵守规范、自律 |
| **传统** | Tradition | 尊重传统、宗教、谦逊 |
| **仁慈** | Benevolence | 帮助亲友、忠诚、诚实 |
| **普遍主义** | Universalism | 平等、公正、保护环境 |

**价值观结构图：**
```
              自我导向
            ↗         ↘
        刺激             普遍主义
       ↙                    ↘
    享乐主义    ←——冲突——→    仁慈
       ↘                    ↙
        成功              传统
          ↘            ↗
           权力 —— 安全 —— 顺从
```

#### 价值观问卷（PVQ简化版中文改编）

**自我导向（3题）：**
1. 独立思考对我来说很重要
2. 我喜欢自己做决定，不受他人影响
3. 自由选择自己的道路对我来说很重要

**刺激（3题）：**
1. 我喜欢冒险和尝试新事物
2. 我需要生活中有变化和刺激
3. 我喜欢挑战自己

**享乐主义（3题）：**
1. 享受生活对我来说很重要
2. 我认为追求快乐是人生的正当目标
3. 我喜欢犒劳自己

**成就（3题）：**
1. 展示自己的能力对我来说很重要
2. 我喜欢设定目标并达成它们
3. 获得认可和成功对我来说很重要

**权力（3题）：**
1. 我希望拥有影响他人的能力
2. 我认为社会地位很重要
3. 我喜欢掌控局面

**安全（3题）：**
1. 生活稳定对我来说很重要
2. 我重视社会秩序和规则
3. 我希望未来是可预测的

**顺从（3题）：**
1. 我认为遵守社会规范很重要
2. 我尽量不给他人添麻烦
3. 我会克制自己的行为以符合社会期望

**传统（3题）：**
1. 我尊重传统和习俗
2. 我认为保持文化传统很重要
3. 我重视宗教或精神信仰

**仁慈（3题）：**
1. 帮助身边的人对我来说很重要
2. 我关心亲友的福祉
3. 我认为忠诚和诚实很重要

**普遍主义（3题）：**
1. 我认为所有人都应该受到平等对待
2. 我关心社会公正
3. 我认为保护环境很重要

### 5.3 约会匹配应用

**价值观匹配策略：**

**核心匹配维度：**
| 价值观对 | 相似重要性 | 说明 |
|----------|------------|------|
| 家庭vs事业 | 高 | 生活重心冲突会引发矛盾 |
| 传统vs开放 | 高 | 生活方式差异大 |
| 安全vs冒险 | 中 | 可互补但需理解 |
| 独立vs亲密 | 中 | 需要平衡空间和连接 |
| 享乐vs成就 | 中 | 可共存，关注优先级 |

**匹配建议：**

1. **核心价值观相似性检查：**
   - 家庭观念（是否有孩子、教育方式）
   - 金钱观（消费、储蓄、投资）
   - 宗教/信仰
   - 人生目标（事业、生活平衡）

2. **价值观层级相似性：**
   - 关注彼此最重视的3-5个价值观是否兼容
   - 对角价值观（如权力vs普遍主义）冲突需要特别关注

3. **可接受差异vs关键匹配：**
   - **必须相似**：家庭观、忠诚、诚实
   - **可互补**：主导性、表达方式、兴趣
   - **需讨论**：宗教、政治、育儿方式

## 6. 恋爱兼容性研究

### 6.1 长期关系预测因子

#### 研究发现（基于Gottman研究所等）

**积极预测因子：**

| 因子 | 预测力 | 说明 |
|------|--------|------|
| **积极互动比例** | 高 | 5:1的积极/消极互动比例预测稳定关系 |
| **人格相似性** | 中高 | 尽责性、宜人性相似最关键 |
| **价值观一致性** | 高 | 核心~3个价值观一致预测长期稳定 |
| **安全依恋** | 高 | 至少一方安全型可显著改善关系 |
| **冲突处理方式** | 高 | 建设性冲突处理预测关系持久 |
| **情感连接** | 高 | Gottman"连接时刻"理论 |
| **共同意义** | 中 | 共同目标、价值观、仪式感 |

**消极预测因子：**

| 因子 | 风险程度 | 说明 |
|------|----------|------|
| **四大骑士** | 极高 | 批评、蔑视、辩解、冷战 |
| **高神经质组合** | 高 | 双方高神经质增加关系不稳定 |
| **焦虑-回避组合** | 高 | 追逐-逃跑循环 |
| **价值观冲突** | 高 | 核心价值观不一致 |
| **信任缺失** | 极高 | 关系基础崩塌 |

### 6.2 婚姻满意度研究

#### 关键发现

**影响婚姻满意度的因素：**

1. **人格因素：**
   - 高尽责性 → 婚姻稳定性↑
   - 高宜人性 → 婚姻满意度↑
   - 高神经质 → 婚姻冲突↑

2. **匹配因素：**
   - 人格相似性 > 人格互补性
   - 价值观一致性是长期稳定最强预测因子
   - 年龄、教育背景相似有利（但不必须）

3. **关系技能：**
   - 沟通质量
   - 冲突解决能力
   - 情感表达能力
   - 同理心

#### 问卷设计建议

**关系预期问卷：**
1. 您对理想伴侣的年龄要求是？
   - 相近（±3岁） / 年长 / 年轻 / 无所谓

2. 您对伴侣教育程度的期望是？
   - 本科及以上 / 相近即可 / 无所谓

3. 您希望未来的家庭角色分工是？
   - 传统分工 / 共同承担 / 根据情况灵活调整

4. 您对生育的期望是？
   - 希望有孩子 / 不想要孩子 / 可以接受没有 / 不确定

**生活方式问卷：**
1. 您的理想周末是？
   - 在家休息 / 户外活动 / 社交聚会 / 文化活动

2. 您的作息习惯是？
   - 早睡早起 / 夜猫子 / 灵活

3. 您的运动习惯是？
   - 每周多次 / 偶尔 / 很少

4. 您对伴侣社交的态度是？
   - 支持对方有独立社交圈 / 希望经常一起社交 / 平衡两者
