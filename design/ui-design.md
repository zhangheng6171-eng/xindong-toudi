## 设计概述

### 产品定位
"心动投递"是一款面向中国年轻人的约会匹配平台，通过66道深度问题问卷，基于价值观、性格、生活方式等多维度匹配，每周为用户推荐高质量的潜在伴侣。

### 核心价值
- **深度匹配**：不是浅层的"看脸"，而是价值观的深度契合
- **慢节奏社交**：每周一次匹配，让用户珍惜每次相遇
- **安全可靠**：真实身份认证，营造健康社交环境
- **温暖陪伴**：不只是工具，更是寻找爱情的伙伴

## 目标用户画像

### 主要用户群体

#### 用户群体A：大学生（20-24岁）
- **特征**：活跃于校园，时间充裕，对爱情充满期待
- **痛点**：社交圈子有限，想找三观契合的人
- **使用场景**：课余时间、宿舍、自习室
- **设备偏好**：手机为主，偶尔用电脑

#### 用户群体B：年轻白领（25-35岁）
- **特征**：工作忙碌，社交圈子固定，重视效率
- **痛点**：没时间社交，相亲太刻意
- **使用场景**：通勤路上、午休时间、晚上
- **设备偏好**：手机为主

### 用户核心需求
1. **真实可靠**：不想遇到"照骗"或虚假信息
2. **高效匹配**：不想浪费时间在无意义的滑动上
3. **隐私保护**：不希望个人信息泄露
4. **有趣体验**：填写问卷不应该是枯燥的
5. **情感期待**：希望找到真正合适的人

## 设计风格与原则

### 设计风格
**现代 · 简洁 · 温暖 · 有活力**

### 核心设计原则

#### 1. 温暖而不甜腻
- 使用柔和的色调，但避免过度粉嫩
- 圆润的形状，但有节制
- 温暖的文案，但不过度煽情

#### 2. 简洁而不单调
- 留白充分，信息层次清晰
- 视觉焦点突出，引导明确
- 微妙的动效增添活力

#### 3. 专业而不冰冷
- 现代感的设计语言
- 人性化的交互细节
- 贴心的提示和引导

#### 4. 活力而不幼稚
- 活泼的色彩点缀
- 流畅的动效
- 但保持成熟的基调

## 配色方案

### 主色调

#### 品牌主色：心动粉
```
主色：#FF6B9D (RGB: 255, 107, 157)
用途：品牌识别、主要按钮、重点强调
```
- 温暖而不俗气
- 代表心动、浪漫
- 区别于传统粉色，更现代

#### 辅助色：柔和紫
```
辅助色：#B794F6 (RGB: 183, 148, 246)
用途：渐变、次要按钮、装饰元素
```
- 增添梦幻感
- 与主色搭配和谐
- 现代感十足

### 功能色

#### 成功/积极
```
#48BB78 (绿色)
用途：匹配成功、完成状态、正向反馈
```

#### 警告/提醒
```
#ED8936 (橙色)
用途：温馨提示、注意事项
```

#### 错误/否定
```
#FC8181 (柔和红)
用途：错误提示、拒绝操作
```

### 中性色

```
深色文字：#2D3748 (主要文字)
次要文字：#718096 (次要信息)
辅助文字：#A0AEC0 (提示文字)
分割线：#E2E8F0 (分割线、边框)
背景色：#F7FAFC (页面背景)
纯白：#FFFFFF (卡片背景)
```

### 渐变色方案

#### 主渐变（品牌渐变）
```
从 #FF6B9D 到 #B794F6
用途：Hero区背景、重要按钮、装饰元素
```

#### 温暖渐变
```
从 #FFF5F7 到 #FFFFFF
用途：卡片背景、区块背景
```

### 配色使用原则

1. **品牌色占比10-20%**：点缀式使用，避免过度
2. **中性色为主**：保证可读性和专业感
3. **渐变用于重点**：Hero区、重要按钮、装饰
4. **功能色克制使用**：只在必要的功能场景

## 字体系统

### 中文字体方案

#### 主标题字体
```
字体：思源黑体 / PingFang SC Medium
字重：Medium (500)
字号：24-32px
行高：1.3
用途：页面标题、卡片标题
```

#### 正文字体
```
字体：思源黑体 / PingFang SC Regular
字重：Regular (400)
字号：14-16px
行高：1.6
用途：正文内容、描述文字
```

#### 辅助字体
```
字体：PingFang SC Light
字重：Light (300)
字号：12-14px
用途：次要信息、提示文字
```

### 英文字体方案

#### 主英文字体
```
字体：SF Pro Display / Inter
字重：Medium (500)
用途：品牌名称、英文标题
```

#### 辅助英文字体
```
字体：Inter
字重：Regular (400)
用途：按钮文字、标签
```

### 字体层级

```
H1 - 页面主标题：32px / Medium / 1.3
H2 - 区块标题：24px / Medium / 1.4
H3 - 卡片标题：18px / Medium / 1.5
Body - 正文：15px / Regular / 1.6
Caption - 辅助文字：13px / Regular / 1.5
Small - 小字：12px / Light / 1.4
```

### 字体使用原则

1. **层级清晰**：通过字号、字重区分信息层级
2. **行高舒适**：中文1.6，标题1.3-1.4
3. **字重克制**：主要用Regular和Medium，Bold少用
4. **可读性优先**：字号不小于12px，重要信息不小于14px

### 2. 注册流程（66道问题问卷）

#### 设计理念
将66道问题分组，每页2-3题，通过精心设计的进度反馈和鼓励文案，让用户保持动力完成问卷。

#### 分组策略
```
第1组：基础信息（3题）

  - 性别
  - 出生年份

  - 学历
  - 职业
  - 城市
  - 感情状态

第3组：生活方式（10题）
  - 作息时间
  - 运动频率
  - 饮食习惯
  - 吸烟饮酒

  - ...

第4组：兴趣爱好（15题）
  - 喜欢的电影类型
  - 音乐偏好
  - 阅读习惯
  - 旅行偏好
  - ...

第5组：性格特质（15题）

  - 冒险/保守
```

#### 设计细节

**进度指示**
- 顶部固定进度条
- 显示当前组别（如"第3组 / 共7组"）
- 进度条使用品牌渐变色
- 完成一组时，进度条平滑动画

**问题卡片**
- 每题独立卡片或区块
- 清晰的题号和问题文字
- 选项使用单选按钮或多选框
- 已选择选项高亮显示
- 选项间距适中（便于点击）

**类型差异化**
- **单选题**：传统radio，但样式美化
- **多选题**：checkbox，"选择所有适用的"
- **量表题**：滑动条或分段选择（如1-10分）
- **排序题**：拖拽排序
- **开放题**：文本输入框

**鼓励元素**
- 每组完成后，显示鼓励语
- "太棒了！你已完成60%"
- "还剩最后一步，加油！"
- 使用emoji或小插画增添趣味

**保存机制**
- 每题选择后自动保存
- 随时可中断，下次继续
- 显示"已自动保存"提示

## 设计规范

### 间距系统

```
间距单位：4px

xs: 4px
sm: 8px
md: 12px
lg: 16px
xl: 24px
2xl: 32px
3xl: 48px

使用原则：
- 组件内部：xs - sm
- 相关元素之间：md - lg
- 独立区块之间：xl - 2xl
- 页面区块之间：2xl - 3xl
```

### 圆角规范

```
组件圆角：
- 按钮：30px（全圆角）
- 卡片：12px
- 输入框：8px
- 标签：16px
- 头像：50%（圆形）
- 照片：8px

图片圆角：
- 小图（<100px）：8px
- 中图（100-300px）：12px
- 大图（>300px）：16px
```

### 阴影规范

```
轻阴影（卡片默认）：
box-shadow: 0 2px 8px rgba(0, 0, 0, 0.08);

中等阴影（悬浮状态）：
box-shadow: 0 4px 16px rgba(0, 0, 0, 0.12);

重阴影（弹窗）：
box-shadow: 0 8px 32px rgba(0, 0, 0, 0.16);
```

### 按钮规范

```
主按钮：
- 高度：44px（移动端），40px（桌面）
- 圆角：30px
- 背景：品牌渐变色
- 文字：白色，15px，Medium
- Hover：轻微放大，加深阴影
- 点击：缩小至98%，反馈明显

次要按钮：
- 高度：44px
- 圆角：30px
- 背景：透明
- 边框：1px solid 品牌色
- 文字：品牌色，15px，Medium

文字按钮：
- 无背景无边框
- 文字：品牌色，15px，Regular
- 下划线hover效果
```

### 输入框规范

```
文本输入框：
- 高度：44px
- 圆角：8px
- 边框：1px solid #E2E8F0
- 内边距：12px 16px
- 字号：15px
- 聚焦时：边框变为品牌色，添加外发光

单选/多选：
- 自定义样式
- 未选中：圆形/方形边框
- 选中：品牌色填充 + 白色对勾
```

### 图标规范

```
图标风格：
- 线性图标为主
- 线宽：2px
- 圆角连接
- 尺寸：24px（默认），20px（小），32px（大）

图标颜色：
- 默认：#718096
- 激活：品牌色
- 可点击：hover时加深
```

### 动效规范

```
页面切换：
- 滑入/滑出动画
- 时长：300ms
- 缓动：ease-out

元素显示/隐藏：
- 淡入/淡出
- 时长：200ms
- 结合位移：translateY(10px) → translateY(0)

按钮反馈：
- 缩放：scale(1) → scale(0.98)
- 时长：100ms

加载动画：
- 品牌色渐变旋转
- 或心跳动画
```

### 响应式断点

```
移动端：< 768px
  - 单列布局
  - 底部导航栏
  - 全屏卡片

平板：768px - 1024px
  - 双列布局
  - 侧边导航栏（可选）
  - 卡片适当放大

桌面：> 1024px
  - 多列布局
  - 左侧导航栏
  - 最大内容宽度：1200px
  - 居中对齐，两侧留白
```

## 无障碍设计

### 色彩对比度
- 正文文字与背景：对比度≥4.5:1
- 大标题与背景：对比度≥3:1
- 避免仅用颜色传达信息

### 可点击区域
- 最小点击区域：44×44px
- 按钮和链接间距充足
- 防止误触

### 文字可读性
- 主要内容字号不小于14px
- 行高不小于1.5
- 段落间距适中

### 键盘导航
- 所有交互元素可通过Tab键访问
- 焦点状态清晰可见
- 支持Enter键激活按钮

### 屏幕阅读器
- 图片添加alt属性
- 图标添加aria-label
- 表单元素关联label
